# Pochette_CD
------
Création de pochette CD avec NFC pour lire de la musique en ligne

Liste des dossiers 
* Matrices (contients les fichiers pour l'impression et la decoupe laser)
* Web (contients les dossiers et les fichiers pour créer sont lecteur audio sur web)
* Tutoriel (contients le fichier pour la créations)

------

## Descriptions détaillé de chaque dossier
### Matrice

Fichier pour la création de la pochette
* Matrice_CD_Impression_V1 plusieur format de fichier.
    * Format AI
    * Format DXF
    * Format EPS
    * Format PNG
    * Format PDF

Fichier pour la création du CD
* Matrice_CD_Laser_V1 plusieur format de fichier.
    * Format AI
    * Format DXF
    * Format EPS
    * Format PNG
    * Format PDF 

Utiliser le logiciel Illustrator ou un équivalent et mettre l'image a la taille voulue puis creer un masque d'écrêtage pour que l'image soit adapter a la forme.
Imprimer le fichier puis le decouper afin de la coller sur la pochette et le CD.
